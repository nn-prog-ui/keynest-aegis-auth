# Nemokey 権限文言

## iOS
- `NSCameraUsageDescription`
  QRコードを読み取り、認証アカウントを登録するためにカメラを使用します。
- `NSFaceIDUsageDescription`
  認証コード、サインイン承認、復元データへのアクセスを保護するために Face ID を使用します。
- `NSUserNotificationsUsageDescription`
  サインイン承認リクエストや重要なセキュリティ通知を受け取るために使用します。

## Android
- `android.permission.CAMERA`
  QRコードを読み取り、認証アカウントを登録するために使用します。
- `android.permission.POST_NOTIFICATIONS`
  サインイン承認リクエストや重要なセキュリティ通知を表示するために使用します。
- `android.permission.USE_BIOMETRIC` / `USE_FINGERPRINT`
  認証コード、サインイン承認、復元データへのアクセスを端末認証で保護するために使用します。
