# Nemokey LP Deployment

## 目的
`web/landing.html` を使って、最小の公開導線をすぐに作る。

## 最短の公開方法
### 1. 静的ホスティングへそのまま置く
- Netlify
- Vercel
- Cloudflare Pages
- GitHub Pages

## 手順
1. `web/landing.html` を静的サイトのトップとして配置
2. ボタンの `mailto:` を仮連絡先から本番連絡先へ置換
3. ストア URL が決まったら CTA を差し替え
4. アナリティクスを入れる

## 先に決めること
- 問い合わせメールアドレス
- App Store URL
- Google Play URL
- 企業導入問い合わせフォーム

## 最初に見る数字
- LP 訪問数
- ストア遷移率
- 問い合わせ数
- デモ依頼数
