# Aegis Auth Privacy Policy

最終更新日: 2026-03-05

## 1. 収集する情報
- 認証アカウント情報（issuer、email、TOTP設定）
- 端末識別子（Push配信用）
- Pushトークン（FCM/APNs）
- ユーザーが入力したクラウドバックアップID

## 2. 利用目的
- ワンタイムコード生成
- サインイン承認通知の配信
- 暗号化バックアップの保存/復元
- セキュリティ向上と不正アクセス防止

## 3. データ保護
- バックアップデータは端末側で暗号化（PBKDF2 + AES-256-GCM）して保存/送信
- 合言葉はサーバーへ平文保存しない
- 生体認証/端末認証でアプリ内データ表示を保護

## 4. 第三者提供
- Push通知配信のため Firebase Cloud Messaging / APNs を利用
- クラウドバックアップ機能でユーザーが指定したサーバーへ暗号化データを送信

## 5. ユーザーの選択
- アカウント削除でローカル保存データを削除可能
- 通知権限はOS設定からいつでも無効化可能
- バックアップを使用しない運用も可能

## 6. 問い合わせ
- 運用チーム連絡先をストアサポートURLに記載すること
