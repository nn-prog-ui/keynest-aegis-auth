# Nemokey Privacy Policy

最終更新日: 2026-04-13

## 1. 収集する情報
- 認証アカウント情報（issuer、account name、TOTP 設定）
- 端末識別子（Push 承認と復元制御用）
- Push トークン（FCM / APNs）
- ユーザーが入力したクラウドバックアップ ID
- セキュリティイベントに関する最小限のメタデータ

## 2. 利用目的
- ワンタイムコード生成
- サインイン承認通知の配信
- 暗号化バックアップの保存 / 復元
- セキュリティ向上と不正アクセス防止
- サポート対応と障害解析

## 3. データ保護
- バックアップデータは端末側で暗号化して保存 / 送信します
- 合言葉や復元用秘密情報は平文でサーバー保存しません
- 生体認証 / 端末認証でアプリ内データへのアクセスを保護します

## 4. 第三者提供
- Push 通知配信のため Firebase Cloud Messaging / APNs を利用します
- クラウドバックアップ機能では、ユーザーが指定した保存先へ暗号化済みデータを送信します
- 法令対応または不正防止が必要な場合を除き、ユーザーデータを第三者へ販売しません

## 5. ユーザーの選択
- アカウント削除でローカル保存データを削除できます
- 通知権限は OS 設定からいつでも無効化できます
- バックアップを使用しない運用も可能です

## 6. 問い合わせ
- ストア掲載のサポート URL またはサポート窓口からお問い合わせください
